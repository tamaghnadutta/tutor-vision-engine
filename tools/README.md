# Tools Directory

This directory contains analysis, debugging, and utility tools.

## Analysis Tools:

- **analyze_bounding_boxes_gpt4v.py** - GPT-4o bounding box analysis
- **analyze_full_solution_areas.py** - Full solution area analysis

## Debug Tools:

- **debug_confidence.py** - Confidence scoring debugging
- **debug_eval_stuck.py** - Evaluation debugging
- **debug_gemini.py** - Gemini model debugging

## Optimization Tools:

- **optimized_gemini_processor.py** - Optimized Gemini processing

## Usage:

Run directly from project root:

```bash
python tools/debug_confidence.py
python tools/analyze_bounding_boxes_gpt4v.py
```

These are primarily development and analysis tools, not part of the main application workflow.
